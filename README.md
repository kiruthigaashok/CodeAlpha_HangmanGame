# 📊 Stock Portfolio Tracker

This is a simple Python-based Stock Portfolio Tracker developed as part of the CodeAlpha Internship.

## 🚀 Project Description
The program allows users to input stock names and quantities, then calculates the total investment value based on predefined stock prices.

---

## ✨ Features
- 📈 Add multiple stocks
- 🔢 Enter quantity for each stock
- 💰 Calculates total investment value
- ❌ Handles invalid stock inputs
- 📁 Saves result to a text file (`portfolio.txt`)

---

## 🛠️ Technologies Used
- Python
- Concepts: Dictionary, Loops, Conditional Statements, File Handling

---

## 📋 How It Works
1. User enters stock name (e.g., AAPL, TSLA)
2. Inputs quantity of shares
3. Program calculates investment value
4. Repeats until user types `done`
5. Displays total investment
6. Saves result to a file
